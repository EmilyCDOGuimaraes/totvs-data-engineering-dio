# Redução de Custos em Farmácias com AWS / AWS Cost Reduction for Pharmacies

Implementation report identifying 3 AWS services to reduce operational costs for a fictional pharmacy chain — Pharma Plus.

Built as part of the **TOTVS Fundamentos de Engenharia de Dados e Machine Learning** bootcamp on DIO.

---

## Services covered

- Amazon EC2 with Auto Scaling — flexible compute, pay for what you use
- Amazon S3 — scalable cloud storage, no hardware costs
- Amazon RDS — managed database, no dedicated admin needed
